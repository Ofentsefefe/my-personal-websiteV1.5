## 📖 About The Project

A modern, fully responsive personal portfolio website that showcases my skills, projects, and professional journey. Built with pure HTML5, CSS3, and JavaScript, this portfolio features stunning visual effects, smooth animations, and a fully functional contact system.

### ✨ **Key Features**

| Feature | Description |
|---------|-------------|
| 🎨 **Modern Design** | Glassmorphism effects, gradient animations, and video backgrounds |
| 📱 **Fully Responsive** | Perfect display on all devices (desktop, tablet, mobile) |
| 🎬 **Interactive Elements** | Hover effects, video previews, and smooth scroll animations |
| 📧 **Contact Form** | Integrated with EmailJS for direct message delivery |
| 🚀 **Performance** | Optimized loading with lazy loading and efficient animations |
| 🔒 **Privacy Compliant** | POPIA compliant with consent checkbox |
| 💾 **Local Backup** | Messages saved locally as backup |
| 🎯 **SEO Ready** | Semantic HTML and meta tags for better visibility |
